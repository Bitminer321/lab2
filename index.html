Whta re conditional Expressions?

In PHP, condition statements are used to make a decision. using if else expressions.

Example:

                
                    $x=30;
                    $z=40;

                    if ($x < $z) {
                        echo "x is less than z.";
                    }
                    else{
                        echo "x is greater than z.";
                    }
